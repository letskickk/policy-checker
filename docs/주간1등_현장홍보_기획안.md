# 주간 1등 후보 현장 홍보 기획안

**"이번 주의 공약왕, 지도부가 직접 찾아갑니다"**

---

## 1. 기획 배경 및 목적

### 배경
개혁신당 정책 멘토링 시스템(policy.reformparty.kr)에서는 출마자가 등록한 공약을 AI가 평가하고, 매주 공약 평균 점수 1위 후보를 **주간 챔피언**으로 선정하고 있다. 현재 주간 챔피언은 랭킹 페이지에 표시되는 것에 그치고 있어, 실질적 보상·동기 부여가 부족한 상태이다.

### 목적
- 주간 1등 후보에게 **당 지도부의 현장 방문 + 유튜브 쇼츠 홍보**라는 실질적 보상 제공
- 출마자들의 공약 품질 향상 경쟁 유도
- 지역 유권자 대상 당·후보 홍보 효과 극대화
- 지도부와 지역 후보 간 연결고리 강화

---

## 2. 운영 구조

### 2-1. 주간 챔피언 선정 기준
| 항목 | 내용 |
|------|------|
| 선정 주기 | 매주 (월요일 00:00 자동 확정) |
| 기준 | 해당 주 점검 완료된 공약의 평균 종합점수 |
| 대상 | 승인(APPROVED) 상태 후보자 중 점검된 공약이 1건 이상인 자 |
| 데이터 소스 | `/api/leaderboard` API · `weekly_champions` 테이블 |

### 2-2. 현장 홍보 프로세스

```
[월요일]          [화~수]              [목~금]            [토~일]
주간 1등 확정 → 지도부 일정 조율   → 현장 방문·촬영  → 편집·배포
               + 후보 사전 인터뷰     + 쇼츠 제작
```

1. **월요일**: 시스템에서 주간 챔피언 자동 확정 → 운영팀이 해당 후보에게 연락
2. **화~수**: 지도부 방문 일정 확정, 후보 사전 인터뷰(공약 핵심 포인트 정리)
3. **목~금**: 지도부가 해당 후보 지역구 방문, 현장 촬영
4. **토~일**: 영상 편집, 유튜브 쇼츠 + SNS 업로드

## 3. 콘텐츠 기획

### 3-1. 유튜브 쇼츠 (메인 콘텐츠)

**포맷**: 세로형 30~60초 쇼츠

**구성안 (A타입 – 지도부 추천형)**
| 시간 | 내용 |
|------|------|
| 0~5초 | 훅: "이번 주 개혁신당 공약왕은?" + 지역구 자막 |
| 5~15초 | 지도부 멘트: "OOO 후보의 공약이 이번 주 1등을 차지했습니다" |
| 15~35초 | 핵심 공약 1~2개 소개 (현장 배경 + 후보 직접 설명) |
| 35~50초 | 지도부 코멘트: 공약의 강점, 왜 좋은 공약인지 |
| 50~60초 | 마무리: 후보 이름·지역구 자막 + 당 로고 |

**구성안 (B타입 – 현장 밀착형)**
| 시간 | 내용 |
|------|------|
| 0~5초 | 훅: "[지역명]에 개혁신당이 왔습니다!" |
| 5~20초 | 지도부 + 후보가 지역구 현장을 함께 걸으며 대화 |
| 20~40초 | 후보가 현장에서 공약 설명 ("여기가 바로 제가 OO하려는 곳입니다") |
| 40~55초 | 지도부 응원 멘트 |
| 55~60초 | 엔딩: 당 슬로건 + 후보 정보 |

### 3-2. 부가 콘텐츠

| 채널 | 포맷 | 내용 |
|------|------|------|
| 인스타그램 릴스 | 15~30초 | 쇼츠 하이라이트 편집본 |
| 당 공식 블로그/홈페이지 | 카드뉴스 | 후보 소개 + 핵심 공약 요약 |
| X(트위터) | 텍스트+사진 | "이번 주 공약왕" 선정 소식 + 현장 사진 |
| 정책 멘토링 사이트 | 배너 | 주간 챔피언 섹션에 쇼츠 영상 임베드 |

### 3-3. 촬영 가이드

- **촬영 인원**: 촬영 1명 + 지도부 + 해당 후보 (최소 3인)
- **장비**: 스마트폰(세로 촬영) + 짐벌 + 핀마이크 2개
- **촬영 장소**: 후보 지역구 내 공약 관련 현장 (예: 재래시장, 학교 앞, 공원 등)
- **분량**: 원본 10~15분 촬영 → 쇼츠 1~2편 + 릴스 1편 편집

## 4. 주간 타임라인

| 요일 | 담당 | 업무 |
|------|------|------|
| **월** | 운영팀 | 주간 챔피언 확인 → 후보 연락 → 축하 안내 |
| **월~화** | 운영팀 | 지도부 일정 확인, 후보와 방문 일정 조율 |
| **화~수** | 운영팀 | 후보 사전 인터뷰 (전화/영상), 촬영 장소·동선 기획 |
| **목~금** | 지도부 + 촬영팀 | 현장 방문, 촬영 진행 |
| **금~토** | 편집팀 | 쇼츠·릴스 편집, 자막·로고 삽입 |
| **일** | 운영팀 | 유튜브·인스타·X 업로드 + 사이트 배너 반영 |

## 5. 기대효과

### 후보 측면
- **공약 품질 경쟁 유도**: 1등 보상이 명확하므로 공약 다듬기에 적극 참여
- **후보 인지도 향상**: 유튜브 쇼츠 노출로 지역 유권자에게 알릴 기회
- **당·후보 유대 강화**: 지도부의 직접 방문이 주는 소속감과 동기 부여

### 당 측면
- **유권자 접점 확대**: 지역 기반 쇼츠 콘텐츠로 자연스러운 당 홍보
- **디지털 홍보 강화**: 쇼츠·릴스 등 숏폼 콘텐츠 누적으로 채널 성장
- **정책 중심 이미지**: AI 공약 점검 + 현장 홍보 = "정책으로 승부하는 당"

### 콘텐츠 측면
- 매주 1편 이상 쇼츠 누적 → 선거 시기 지역별 콘텐츠 라이브러리 확보
- 후보별 지역 특색 있는 현장 콘텐츠 축적

## 7. 유의사항 및 리스크 관리

### 공정성
- 선정 기준은 시스템 자동 계산 (평균 점수)이므로 자의적 개입 불가
- 동점 시 점검 공약 수가 많은 후보 우선 (기존 로직 유지)
- 주간 챔피언 이력은 DB에 기록되어 투명하게 조회 가능

### 선거법 준수
- 선거운동 기간 전: "정책 소개" 중심으로 구성, 직접적 투표 호소 금지
- 선거운동 기간: 선거법 허용 범위 내에서 온라인 홍보물로 활용
- 콘텐츠 게시 전 법무팀(또는 선거법 전문가) 사전 검토 필수

### 운영 리스크
- **1등 후보 불참 시**: 2등 후보에게 기회 제공 또는 해당 주 건너뛰기
- **지도부 일정 불가 시**: 차주로 이월하거나, 다른 당 인사(국회의원 등)가 대체 방문
- **지역 접근성**: 원거리 지역의 경우 화상 인터뷰 + 현지 촬영팀 활용 고려

### 콘텐츠 품질
- 촬영·편집 가이드라인 사전 수립
- 후보 사전 인터뷰를 통해 핵심 메시지 정리 후 촬영 진행
- 최소 편집 기준: 자막, 당 로고, 후보 정보 삽입

## 8. 시스템 연동 (향후 확장)

현재 정책 멘토링 시스템과 연동할 수 있는 부분:

| 기능 | 설명 | 난이도 |
|------|------|--------|
| 챔피언 알림 자동화 | 주간 챔피언 확정 시 운영팀에 자동 이메일/슬랙 알림 | 낮음 |
| 쇼츠 임베드 | 리더보드 페이지에 해당 주 쇼츠 영상 링크 표시 | 낮음 |
| 후보 프로필 연동 | 지도 페이지 후보 상세에 "주간 1등 홍보 영상" 배지·링크 추가 | 중간 |
| 홍보 아카이브 | 역대 주간 챔피언 + 쇼츠 영상 모아보기 페이지 | 중간 |

## 요약

> 매주 공약 점수 1등 후보의 지역구에 지도부가 직접 방문하여
> 유튜브 쇼츠를 제작·배포하는 프로그램.
> 후보에게는 홍보 보상, 당에게는 콘텐츠 축적과 지역 접점 확대 효과.
